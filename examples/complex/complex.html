Rendering small HTML documents without owning the drawing stack

April 29, 2026 · Layout demo · 4 min read

A lightweight renderer is useful when an app needs rich text, release notes, tooltips, or help pages without embedding a full browser.

这个段落使用微软雅黑作为默认字体，同时保留 English words 来测试混排 fallback。 This example uses golitehtml to parse HTML, apply a compact CSS subset, wrap text, measure images, and emit backend-neutral drawing commands. The final SVG is painted by a tiny custom Canvas implementation.

The key integration point is small: provide text metrics and implement three draw callbacks for rectangles, text, and images. 当前块指定 Microsoft YaHei。

The current demo exercises:

  • descendant selectors, classes, ids, and inline emphasis
  • margins, padding, borders, colors, and inherited fonts: Consolas
  • line wrapping, explicit image sizing, and SVG output
