Rendering small HTML documents without owning the drawing stack

April 29, 2026 · Layout demo · 4 min read

A lightweight renderer is useful when an app needs rich text, release notes, tooltips, or help pages without embedding a full browser.

This example uses golitehtml to parse HTML, apply a compact CSS subset, wrap text, measure images, and emit backend-neutral drawing commands. The final SVG is painted by a tiny custom Canvas implementation.

The key integration point is small: provide text metrics and implement three draw callbacks for rectangles, text, and images.

The current demo exercises:

  • descendant selectors, classes, ids, and inline emphasis
  • margins, padding, borders, colors, and inherited fonts
  • line wrapping, explicit image sizing, and SVG output
